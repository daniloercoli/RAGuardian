RAG personale

Per uffici dove non basta una singola knowledge base

Multi-utente, ogni dipendente ha documenti, email, Microsoft Drive, data sources, conversazioni e vector index isolati. L'azienda mantiene provider, policy e deployment sotto controllo.

👤

Workspace per utente

Ogni account ha upload, FileIndex, data sources, conversazioni e collection Chroma dedicate. Le query non attraversano il confine dell'utente.

✉️

Email personali

Il connettore IMAP indicizza mailbox, thread e allegati supportati nel RAG dell'utente proprietario. Le credenziali sono cifrate o referenziate via variabile ambiente.

☁️

Microsoft Drive

Il plugin Microsoft Drive legge PDF, TXT e Markdown da OneDrive o SharePoint, li materializza nello snapshot del workspace e li indicizza nel perimetro corretto.

🔑

API key per workspace

Plugin WordPress, portali e integrazioni server-to-server usano API key legate a uno specifico utente. Una chiave non può interrogare il RAG di un altro.

Perché noi

Mentre altri offrono un chatbot, noi offriamo un'infrastruttura

RAGuardian non è un semplice wrapper LLM. È un'infrastruttura RAG completa, progettata per le esigenze delle aziende che non possono permettersi di compromettere la sicurezza dei dati.

🏛️

Sovranità dei dati

Con Regolo.ai, ogni inference gira su infrastrutture europee. I tuoi prompt e risposte non vengono mai salvati. Zero data retention significa zero rischi di data leakage.

🔄

Scambia provider in un click

Oggi Regolo.ai, domani Mistral, dopodomani un modello locale. Nessuna modifica al codice. Il tuo team sceglie il giusto equilibrio tra privacy, costi e performance.

🎙️

Audio-native

Non solo PDF. Carica registrazioni riunioni, podcast interni o chiamate sales. Vengono trascritte e indizzate automaticamente. I dipendenti possono interrogare la knowledge base parlando.

🔌

Integration-ready

WordPress plugin con chatbot, sincronizzazione articoli e shortcode, REST API versionata, API keys per workspace con scope granulari, streaming NDJSON. Integriamo con il tuo stack, non il contrario.

Plugin WordPress

Trasforma qualsiasi sito in un'esperienza RAG

Il nostro plugin WordPress open source collega il tuo sito pubblico a un workspace RAGuardian dedicato. Solo server-to-server. La API key non raggiunge mai il browser.

💬

Chatbot fluttuante & shortcode inline

Widget completamente personalizzabile (posizione, colori, avatar, CSS) o shortcode inline [rag_chat] con overrides per pagina. Pulsante TTS, upload audio, visibilità delle fonti — tutto configurabile.

📰

Ingestione articoli & sincronizzazione live

Importa i tuoi articoli pubblici da un export WXR, poi mantienili sincronizzati. Ogni pubblicazione, aggiornamento o eliminazione fluisce automaticamente in RAGuardian. Niente re-indexing manuale.

🔒

Zero esposizione della chiave

Client PHP server-side. La API key vive nelle opzioni WordPress, usata solo per chiamate autenticate. Le richieste AJAX sono proxyate attraverso PHP — mai esposte a JavaScript.

🧪

Pronta per produzione

Health check, rate limiting server-side, client context (titolo pagina, URL, locale), retry con backoff esponenziale. Suite E2E completa con Playwright. Licenza MIT.
