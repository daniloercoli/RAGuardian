Modello Locale

Esegui LLM sul tuo server. Nessun dato esce mai dal tuo network. Massima privacy, zero dipendenze esterne.

✓
Privacy totale Zero data transfer: i dati non lasciano mai il tuo datacenter
✓
Modalità air-gapped Funziona senza connessione internet
✓
OpenAI-compatible Collega Ollama, LM Studio, vLLM o qualsiasi endpoint compatibile
⚠
Requisiti hardware Servizio con GPU per modello di buona qualità. Embedding locale richiede ~2GB RAM minimo.
✓
Costo Nessun costo API: una volta configurato, i costi sono solo infrastruttura
✓
Embedding locale sentence-transformers: nessuna dipendenza cloud per embeddings

Esempio di configurazione: Ollama con Qwen 3.5 122B sul tuo server. Embedding locale con sentence-transformers. Niente dati esce mai dal tuo network. Compliance massima.

Configurazione

Cambia provider in 30 secondi

1. Apri la dashboard admin

Vai a /admin/config, seleziona provider e modello per l'istanza. La scelta vale come policy centrale per tutti i workspace.

2. Imposta le credenziali server

Per Regolo: REGOLO_API_KEY. Per Mistral: MISTRAL_API_KEY. Per locale: OPENAI_BASE_URL=localhost:11434. Le API key dei singoli utenti restano separate.

3. Pronto

Le chat e i connettori continuano a usare il workspace personale dell'utente. Nessuna migrazione dati necessaria.
