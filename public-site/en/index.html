Personal RAG

For offices where one knowledge base is not enough

Every employee gets isolated documents, email, Microsoft Drive, data sources, conversations, and more. The company keeps providers, policy, and deployment under control.

👤

Workspace per user

Each account has dedicated uploads, FileIndex, data sources, conversations, and Chroma collection. Queries do not cross the user's boundary.

✉️

Personal email

The IMAP connector indexes mailbox messages, threads, and supported attachments into the owning user's RAG. Credentials are encrypted or referenced through environment variables.

☁️

Microsoft Drive

The Microsoft Drive plugin reads PDF, TXT, and Markdown from OneDrive or SharePoint, materializes a workspace snapshot, and indexes it within the correct boundary.

🔑

Workspace API keys

WordPress plugins, portals, and server-to-server integrations use API keys tied to a specific user. One key cannot query another person's RAG.

WordPress Plugin

Turn any site into a RAG-powered experience

Our open-source WordPress plugin connects your public site to a dedicated RAGuardian workspace. Server-to-server only. The API key never reaches the browser.

💬

Floating chatbot & inline shortcode

Fully customizable chat widget (position, colors, avatar, CSS) or inline [rag_chat] shortcode with per-page overrides. TTS button, audio upload, source visibility — all configurable.

📰

Article ingestion & live sync

Import your public articles from a WXR export, then keep them synchronized. Every publish, update or delete flows to RAGuardian automatically. No manual re-indexing.

🔒

Zero key exposure

Server-side PHP client. The API key lives in WordPress options, used only for authenticated calls. AJAX requests are proxied through PHP — never exposed to JavaScript.

🧪

Production-grade

Health checks, server-side rate limiting, client context (page title, URL, locale), exponential backoff retry. Full E2E test suite with Playwright. MIT licensed.
