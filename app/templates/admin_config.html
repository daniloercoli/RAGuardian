ReRanking

None uses Chroma's top candidates as-is. Source Diversity prevents one document from filling the reranker pool. MMR Diversity balances Chroma relevance with semantic variety before reranking.

Note: With ReRanking enabled, Candidate Diversity runs before reranking. The final documents passed to the LLM are controlled by the "Retrieval Results (k)" parameter above.

Voice & Audio

Select an OpenAI-compatible speech provider for audio transcription and text-to-speech responses. Built-in defaults come from the provider file; custom providers can be added below.

STT is used by POST /api/v1/audio. TTS is used by POST /api/v1/tts. Leave STT Language empty for provider autodetect, or set an ISO code such as it to force transcription language. Audio endpoints require API keys with ingest or speech scope.

OCR

OCR is used for images and as fallback when uploaded PDFs cannot be parsed into text. Regolo OCR with deepseek-ocr-2 is available by default; add custom providers below when needed.

OCR fallback runs only after normal PDF parsing returns no text. Image uploads from chat use OCR directly.
